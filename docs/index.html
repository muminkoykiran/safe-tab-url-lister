Privacy

What we collect: nothing

Safe Tab URL Lister makes zero network requests. All processing happens locally in your browser. No data is ever transmitted, stored on a server, or shared with third parties.

Permission Why it's needed Sends data externally?
tabs Read the URL and title of each open tab so they can be listed and copied. No
Clipboard write Place the formatted output on your clipboard when you press Copy. No

Only the tabs permission is declared in the manifest. No host permissions, no storage, no background scripts.

FAQ

Frequently asked questions

How do I copy all my open Chrome tab URLs at once?
Install Safe Tab URL Lister, click the extension icon, and press Copy. All tab URLs from the current window are on your clipboard immediately.
Can I export Chrome tabs as Markdown links?
Yes. Select "Markdown links" from the format dropdown. The output will be [Page Title](URL) entries, one per tab, ready to paste into Obsidian, Notion, GitHub, or any Markdown editor.
Is there a Chrome extension that exports tabs as JSON?
Yes. Select the JSON format. The output is a JSON array where each element has a title and url field.
Does this extension send my data anywhere?
No. There are zero network requests. Your URLs never leave your device.
Can I export tabs from all Chrome windows at once?
Yes. Enable the "Include all windows" checkbox to collect tabs from every open Chrome window simultaneously.
Does it work offline?
Yes, completely. No external services required.
